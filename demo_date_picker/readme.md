###带有基本功能的 date-picker 控件

在 HTML 中插入` <input type='text' id='date-picker'>` <br>

引入 css <link rel="stylesheet" href="calendar.css"><br>

引入 js <script src="calendar.js"></script><br>

即可使用 date-picker 控件<br>
